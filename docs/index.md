# eShopping BigCommerce Theme User Guide

Welcome to the official documentation for the **eShopping** BigCommerce theme by [PapaThemes](https://papathemes.com).

eShopping is a modern, feature-rich theme designed for B2B and B2C stores. It includes four industry-ready presets, a powerful sidebar navigation, bulk order mode, cart goal bars, marketing popups, and over 85 configurable settings -- all manageable from the Theme Editor without any coding.
