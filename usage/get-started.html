    Get Started¶

    Thank you for choosing the eShopping BigCommerce theme! This guide will walk you through the initial setup so you can get your store looking great as quickly as possible.

    What You Need¶

    To set up the theme exactly like our demo stores, you will need to install the free PapaThemes Widgets app. This app adds extra drag-and-drop widgets to Page Builder so you can visually build content without any coding.

    Opening the Theme Editor¶

    All theme customization happens in the Theme Editor. To open it:

    1. Log in to your BigCommerce admin panel.
    2. Go to Storefront > My Themes.
    3. Find the eShopping theme and click the Customize button.

    This opens Page Builder with the Theme Editor panel on the left side.

    BigCommerce admin panel showing Storefront > My Themes with the Customize button highlighted on the eShopping theme

    Customizing Typography, Fonts, and Colors¶

    To adjust your store's fonts, text sizes, colors, and button styles:

    1. In the Theme Editor, go to Theme Styles > General.
    2. Browse the available options for typography, font families, colors, and button appearance.
    3. Make your changes and click Save when you are happy with the result.

    There are dozens of settings here to help you create a unique look for your brand.

    Theme Styles > General panel showing typography, font, and color options

    Customizing the Checkout Page¶

    To style the checkout page's typography and colors:

    1. In the Theme Editor, go to Theme Styles > Optimized Checkout.
    2. Adjust the available color and font settings.
    3. Click Save.
    Theme Styles > Optimized Checkout panel showing checkout-specific styling options

    eShopping-Specific Settings¶

    The eShopping theme adds a dedicated eShopping section to the Theme Editor with over 85 configurable settings. This is where you control all the features that make eShopping unique.

    To access these settings:

    1. In the Theme Editor, go to Theme Styles > eShopping.

    2. You will see settings organized into the following groups:

      • Fonts -- Monospace font for prices, SKUs, and codes
      • Colors -- Full color palette (Terra, Forest, Bark, Cream, White)
      • Homepage Sections -- Toggle hero carousel, trust strip, featured products, bestsellers, new arrivals, categories, newsletter
      • Trust Strip -- Customize trust badge labels
      • Header -- Topbar and navigation bar colors, social icons, welcome text
      • Footer -- Footer background, text, heading, link colors
      • Search -- Keyword suggestions, voice search, animated search placeholder
      • Sidebar -- Sidebar display options
      • Sidebar Promo Card -- Promotional text in the sidebar
      • Products by Category -- Category tab sections on the homepage
      • Product Page (PDP) -- Shipping text, mobile sticky add-to-cart, FAQ tab, warranty text
      • Frequently Bought Together -- Product count and visual discount percentage
      • Cart Cross-sell -- Cross-sell product recommendations in cart
      • Bulk Order -- Toggle bulk order mode on category and brand pages
      • Cart Goal Bar -- Configurable spending goal tiers
      • Urgency Signals -- Live view count and last order time displays
      • Recent Sales Popup -- Toast notifications about recent purchases
      • Newsletter Popup -- Newsletter signup popup
      • Promotion Popup -- Promotional offer popup
      • Exit Intent Popup -- Popup triggered when visitors are about to leave

    3. Adjust any setting and click Save.

    Theme Styles panel showing the eShopping section expanded with its setting groups listed

    Installing the PapaThemes Widgets App¶

    The PapaThemes Widgets app adds custom widgets to Page Builder, giving you more options for building your store's content visually.

    1. Find and install PapaThemes Widgets from the BigCommerce Apps Marketplace.
    2. After installation, go to Apps > My Apps in your admin panel's left sidebar.
    3. Click the Launch button next to PapaThemes Widgets to open the app.
    BigCommerce admin panel showing Apps > My Apps with the PapaThemes Widgets Launch button
    1. In the app, click the Install buttons to install all the available widgets.
    PapaThemes Widgets app interface showing Install buttons for each available widget
    1. Return to the Theme Editor (Storefront > My Themes > Customize). You should now see the PapaThemes widgets listed in the Page Builder sidebar.
    Page Builder sidebar showing the PapaThemes custom widgets available for use

    Selecting a Preset Style¶

    The eShopping theme ships with four industry preset styles designed to give you a head start. Each preset customizes the theme's colors, typography, and layout to match a specific industry, so your store looks polished right away.

    To select a preset:

    1. In the Theme Editor, go to Theme Styles.
    2. At the top of the panel, use the style selector dropdown to choose from the available presets.
    3. Click Save to apply the preset.

    Industrial¶

    A bold, utilitarian color palette with dark tones and strong accents. Best suited for stores selling tools, hardware, safety equipment, and industrial supplies.

    eShopping storefront with the Industrial preset applied -- dark header, bold accent colors, industrial product imagery

    AutoParts¶

    A high-contrast color scheme with automotive-inspired accents. Designed for stores selling auto parts, accessories, and vehicle maintenance products.

    eShopping storefront with the AutoParts preset applied -- automotive color scheme, parts-oriented product layout

    Packaging¶

    Clean, professional tones with neutral colors. Tailored for stores selling packaging supplies, shipping materials, and bulk packaging products.

    eShopping storefront with the Packaging preset applied -- clean neutral tones, packaging product imagery

    Electronics¶

    A modern, tech-forward color palette with cool tones and sharp accents. Built for stores selling electronics, gadgets, components, and tech accessories.

    eShopping storefront with the Electronics preset applied -- modern tech color scheme, electronics product layout

    After selecting a preset, you can further customize any individual setting in Theme Styles > eShopping to fine-tune the look and feel for your brand.
